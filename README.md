# KJFM papers edit

KJFM(한국프랜차이즈경영학회지) 논문을 칸에 붙여넣거나 기존 워드를 올려 **검증된 KJFM 양식 .docx**로 내보내는 **브라우저 웹앱**. 서버 없이 브라우저 안에서 동작합니다.

## ▶ 바로 쓰기
**https://ywhancontact-ai.github.io/kjfm-papers/**

링크만 열면 됩니다 (설치·로그인 불필요). 첫 로딩에 ~20초(브라우저용 파이썬 엔진 다운로드), 이후 빠름.

## 기능
- 제목·저자·초록·본문·표·그림·참고문헌·부록 입력 → **Export to Word**
- 기존 `.docx` 업로드 → 자동 분해해 칸 채움
- 머릿글(홀/짝, 권호 17(2)), DOI 자동 연동, 2단 레이아웃, APA 참고문헌, 각주
- **Check refs** — 본문 인용 ↔ 참고문헌 대조
- 미리보기(레이아웃 근사)

## 작동 원리 (양식이 깨지지 않는 이유)
워드 생성 엔진을 JavaScript로 다시 짠 게 아니라, **검증된 파이썬 엔진(`engine.py`)을 브라우저에서 그대로 실행**(Pyodide)합니다. 서버판과 **출력이 byte 단위로 동일**함을 12개 논문으로 검증했습니다.

## 구성
| 파일 | 역할 |
|------|------|
| `index.html` | 입력 폼 UI + Pyodide 부트스트랩 |
| `engine.py` | KJFM 양식 .docx 생성·파싱 엔진 (브라우저에서 실행) |
| `template.docx` | 빈 KJFM 양식 템플릿(내용 없음) |
